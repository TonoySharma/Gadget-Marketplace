# Project Name: Gadget Marketplace(Mobile Shop)

A powerful and scalable backend API for the **Gadget Marketplace** application. 
This server provides RESTful APIs to manage gadgets, search products, filter data, pagination, featured products,
 and CRUD operations using **Node.js, Express.js, TypeScript, and MongoDB**.


## 🌐 Live Server

> https://gadget-hub-theta.vercel.app/

## ✨ Features

- 🔐 RESTful API Architecture
- 📦 Add New Products
- 🛍️ Browse All Products
- 🔍 Search Products by Title & Brand
- 🗂️ Filter Products by Category
- 💰 Filter Products by Price Range
- 📄 Pagination Support
- ⭐ Featured Products API
- 📱 Product Details API
- 🗑️ Delete Product
- 📈 Sorting Support
- 🌐 MongoDB Atlas Integration
- ⚡ TypeScript Support
- 🔄 CORS Enabled
